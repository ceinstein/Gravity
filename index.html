Gravity

Small gravity simulator

View the Project on GitHub ceinstein/Gravity

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

To compile:

javac Gravity.java Space.java Player.java Planet.java Vec2D.java UserInput.java

To run:

java Gravity (rows) (cols)

User Input:

 - w (Thrusts ship down)
 - s (Thrusts ship up)
 - a (Thrusts ship right)
 - d (Thrusts ship left)
